# Soil Testing Admin Panel - Frontend

Production-ready React Vite Admin Panel for Soil Testing / Site Resource Management System.

## Features

- Role-based dashboards (SuperAdmin, Admin, User)
- JWT authentication with protected routes
- Responsive design with Tailwind CSS
- Modern UI components
- API integration with Axios

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
